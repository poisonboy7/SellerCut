# ✂️ [SeelerCut] - 쇼핑몰 상세페이지 원클릭 리사이즈 & 분할 툴

> **"포토샵 없이 클릭 한 번으로 끝내는 오픈마켓 규격 맞춤 서비스"**
> 본 서비스는 국내 쇼핑몰 셀러분들이 상품 등록 시 겪는 상세페이지 규격 맞춤 및 이미지 분할의 번거로움을 해결하기 위해 제작된 100% 무료 웹 도구입니다.

## ✨ 핵심 특장점
1. **서버 비용 ZERO / 보안 완벽:** 업로드하신 이미지는 외부 서버로 절대 전송되지 않으며, 사용자의 브라우저(Client-side) 내에서 HTML5 Canvas API를 통해 안전하게 처리됩니다.
2. **회원가입 및 결제 없음:** 복잡한 절차 없이 접속하자마자 바로 사용하실 수 있습니다.
3. **국내 주요 플랫폼 규격 완벽 반영:** 버튼 하나로 각 마켓 최적화 사이즈 변환 및 업로드 에러 방지 분할을 지원합니다.

## 📊 지원 규격 리스트
* **네이버 스마트스토어 / G마켓 / 옥션 / 롯데ON:** 가로 860px 리사이즈 + 세로 4000px 자동 분할
* **쿠팡 (Coupang):** 가로 780px 리사이즈 + 세로 3000px 자동 분할 (GIF 자동 제외 및 JPG/PNG 강제 변환)
* **11번가 / 도매꾹:** 가로 860px 리사이즈 + 세로 3000px 분할 및 용량 최적화
* **무신사 / 오늘의집:** 가로 850px 리사이즈 + 세로 4000px 자동 분할
* **카카오 쇼핑하기:** 가로 750px 리사이즈 + 세로 3000px 자동 분할
* **오픈마켓 통합 썸네일:** 중앙 기준 1:1 정방형 크롭 및 1000x1000px 변환

## 🛠️ 기술 스택
* HTML5 / CSS3 (Responsive Design)
* Vanilla JavaScript (Canvas API)
* JSZip (Client-side ZIP Archive Generation)
